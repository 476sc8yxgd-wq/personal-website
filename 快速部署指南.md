## 🔄 备用方案

如果自动脚本无法使用，选择以下方式：

### 方案 A：WinSCP（图形化）

1. 下载: https://winscp.net/
2. 连接: `root@43.138.3.207`
3. 上传: `frontend/dist/*` → `/var/www/personal-website/public/`
4. SSH 重启: `ssh root@43.138.3.207 'cd /var/www/personal-website && docker-compose restart'`

### 方案 B：命令行（Mac/Linux）

```bash
scp -r frontend/dist/* root@43.138.3.207:/var/www/personal-website/public/
ssh root@43.138.3.207 'cd /var/www/personal-website && docker-compose restart'
```

## 📞 遇到问题？

### 问题：无法连接服务器

检查：
1. 服务器 IP: 43.138.3.207
2. 用户名: root
3. 密码是否正确
4. 网络连接是否正常

### 问题：部署后无法访问

检查：
```bash
# SSH 登录服务器
ssh root@43.138.3.207

# 检查容器状态
cd /var/www/personal-website
docker ps

# 查看日志
docker logs personal_website

# 重启容器
docker-compose restart
```

### 问题：网站样式异常

1. 清除浏览器缓存 (Ctrl + Shift + Delete)
2. 使用无痕模式访问
3. 检查 assets 目录是否正确上传
