# 前端页面修复指南

## 问题描述
服务器上的"关于我"页面显示的是旧版本，需要更新为带有双列布局和王者荣耀省标的新版本。

## 解决方法

### 方法一：使用SCP上传文件（推荐）

1. 打开PowerShell，进入项目目录：
```powershell
cd C:\Users\28945\Desktop\个人网站重制版
```

2. 执行以下命令上传About.tsx到服务器：
```powershell
scp -o StrictHostKeyChecking=no frontend\src\pages\About.tsx root@43.138.3.207:/root/个人网站重制版_20251226022228/frontend/src/pages/About.tsx
```

3. 如果提示输入密码，输入：`Tencent@12345`

### 方法二：使用PuTTY上传文件

如果你安装了PuTTY：

1. 使用PuTTY的pscp工具上传：
```powershell
"C:\Program Files\PuTTY\pscp.exe" -pw Tencent@12345 frontend\src\pages\About.tsx root@43.138.3.207:/root/个人网站重制版_20251226022228/frontend/src/pages/About.tsx
```

### 方法三：使用SSH手动创建文件

1. 使用SSH登录到服务器：
```powershell
ssh root@43.138.3.207
```
密码：`Tencent@12345`

2. 登录后，使用vi或nano编辑文件：
```bash
cd /root/个人网站重制版_20251226022228/frontend/src/pages
vi About.tsx
```

3. 将本地的 `C:\Users\28945\Desktop\个人网站重制版\frontend\src\pages\About.tsx` 文件内容复制粘贴进去

## 上传文件后的部署步骤

文件上传成功后，执行以下命令重新构建和部署：

```bash
# 进入前端项目目录
cd /root/个人网站重制版_20251226022228/frontend

# 重新构建前端
npm run build

# 复制构建文件到后端容器
docker cp dist/. personal_website_backend:/app/public/

# 重启后端容器
docker restart personal_website_backend
```

## 常见问题

### Q: SCP命令不工作怎么办？
A: 请先安装OpenSSH客户端，或使用方法二（PuTTY）

### Q: SSH连接不上怎么办？
A: 检查服务器IP和端口，确保网络通畅

### Q: 构建失败怎么办？
A: 确保node_modules已安装，执行 `npm install` 后再试

## 技术支持

如果遇到问题，请检查：
- 服务器后端容器状态：`docker ps`
- 后端容器日志：`docker logs personal_website_backend`
- 前端构建日志：查看npm run build的输出
