## 🚀 部署到服务器

### 方式一：使用自动部署脚本（推荐）

运行以下命令：
```powershell
.\deploy-frontend.ps1
```

**说明**：
1. 首次运行可能需要输入服务器密码
2. 脚本会自动：
   - 清理服务器旧文件
   - 上传新的前端文件
   - 重启 Docker 容器
   - 验证服务状态

### 方式二：使用 WinSCP（图形化）

1. 下载 WinSCP: https://winscp.net/
2. 连接信息：
   - 主机: 43.138.3.207
   - 用户: root
   - 密码: [您的服务器密码]
3. 上传文件：
   - 本地: `frontend/dist/*`
   - 远程: `/var/www/personal-website/public/`
4. SSH 登录重启服务：
   ```bash
   ssh root@43.138.3.207
   cd /var/www/personal-website
   docker-compose restart
   ```

### 方式三：手动上传

如果上述方法都不可用，可以：

1. 使用其他 FTP/SFTP 工具上传文件
2. 手动执行服务器命令：
   ```bash
   ssh root@43.138.3.207
   cd /var/www/personal-website/public
   rm -rf *
   exit
   ```
3. 上传 `frontend/dist` 目录的所有文件到 `/var/www/personal-website/public/`
4. 重启服务：
   ```bash
   ssh root@43.138.3.207
   cd /var/www/personal-website
   docker-compose restart
   ```
